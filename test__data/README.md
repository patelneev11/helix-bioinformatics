# Test Data for helix

## Insulin Gene Sequences

Sequences used for testing the GeneCompare tool

- `human_insulin_mrna.fasta` - Homo sapiens insulin mRNA
  - Source: NCBI RefSeq NM_000207
  - Downloaded: 1/1/2026
 
- `mouse_insulin_mrna.fasta` - Mus musculus insulin mRNA
  - Source: NCBI RefSeq NM_001185083
  - Downloaded: 1/1/2026

- `dog_insulin_mrna.fasta` - Canis lupus familiaris mRNA
  - Source: NCBI RefSeq NM_001130093.2
  -  Downloaded: 1/1/2026

## Why these sequences?

Insulin is highly conserved across mammals, therefore making it perfect for evolutionary comparison and demonstrating sequence alignment.
