# Helix: Bioinformatics tool for student researchers

## What is Helix?
A suite of Python-based tools to help high school and college students analyze genetic data for their research projects.

## Planned Tools
- **GeneCompare**: DNA sequence comparison
- **MutationMapper**: Variant visualization and analysis
- **ProteinPredict**: Protein structure prediction

## Status
Currently in the development stage. Started January 2026.

## About
Built by Neev Patel, started as a passion project to make bioinformatics accessible to everybody
